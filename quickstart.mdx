## Installation

1. Import the SendUnitySDK package into your Unity project
2. Add the `SendSDKManager` component to a GameObject in your scene
3. Configure your RPC endpoint and program IDs in the inspector

## Quick Start

### Setting Up NPCs

<img src="/images/npc-manager.png" alt="NPC Manager" />

1. Create a new GameObject and add the `NPCManager` component
2. Open the Web3 NPC Manager (Window > SendUnitySDK > Web3 NPC Manager)
3. Add a new NPC:

   ```csharp
   var npcState = new NPCState
   {
       npcId = "quest_giver_1",
       contractAddress = "YOUR_CONTRACT_ADDRESS",
       programId = "YOUR_PROGRAM_ID",
       interactionType = NPCInteractionType.QuestGiver,
       isInteractable = true,
       availableActions = new[] { "startQuest", "completeQuest" }
   };
   npcManager.RegisterNPC(npcState);
   ```

4. Add the `Web3NPC` component to your NPC GameObject and select the registered NPC

### Handling NPC Interactions

```csharp
// In your game script
public class QuestManager : MonoBehaviour
{
    [SerializeField] private NPCManager npcManager;

    public async void StartQuest()
    {
        if (!SendSDKManager.Instance.IsAuthenticated())
        {
            Debug.LogError("Please login first!");
            return;
        }

        bool success = await npcManager.InteractWithNPC(
            "quest_giver_1",
            "startQuest",
            new object[] { "QuestID123", "PlayerID456" }
        );

        if (success)
        {
            Debug.Log("Quest started successfully!");
        }
    }
}
```

### Managing NFTs

<img src="/images/asset-manager.png" alt="Asset Manager" />

```csharp
public class ItemManager : MonoBehaviour
{
    [SerializeField] private GameNFTHandler nftHandler;

    public async void MintGameItem(string metadataUri)
    {
        if (!SendSDKManager.Instance.IsAuthenticated())
        {
            Debug.LogError("Please login first!");
            return;
        }

        await nftHandler.MintNFT(metadataUri);
    }
}
```
