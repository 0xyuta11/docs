## üß∞ Setup Guide

### 1. Create the Dialogue UI

1. **Create a Canvas**:  
   - `UI ‚Üí Canvas`
   - Set `Render Mode ‚Üí Screen Space - Overlay`

2. **Create a Dialogue Panel**:  
   - `UI ‚Üí Panel` ‚Üí Name it `DialoguePanel`  
   - Set its visibility to **inactive** by default.

3. **Create a Text Component** inside the `DialoguePanel`:
   - `UI ‚Üí Text` or `TextMeshPro - Text` ‚Üí Name it `DialogueText`
   - Resize and style the text box as needed.

---

### 2. Create the Dialogue System

**Script**: `DialogueSystem.cs`  
Attach it to an empty GameObject (e.g. `DialogueManager`).

```csharp
using UnityEngine;
using UnityEngine.UI;

public class DialogueSystem : MonoBehaviour
{
    public GameObject dialoguePanel;
    public Text dialogueText;

    private string[] dialogueLines;
    private int currentLine;

    public static DialogueSystem instance;

    void Awake()
    {
        instance = this;
        dialoguePanel.SetActive(false);
    }

    public void StartDialogue(string[] lines)
    {
        dialogueLines = lines;
        currentLine = 0;
        dialoguePanel.SetActive(true);
        ShowLine();
    }

    void Update()
    {
        if (dialoguePanel.activeInHierarchy && Input.GetKeyDown(KeyCode.E))
        {
            currentLine++;
            if (currentLine < dialogueLines.Length)
                ShowLine();
            else
                EndDialogue();
        }
    }

    void ShowLine()
    {
        dialogueText.text = dialogueLines[currentLine];
    }

    void EndDialogue()
    {
        dialoguePanel.SetActive(false);
    }
}
```

> üìù Make sure to assign both `DialoguePanel` and `DialogueText` in the Inspector.

### 3. Create the NPC Interaction Script

**Script**: `NPCInteraction.cs`  
Attach this to any NPC character GameObject.

```csharp
using UnityEngine;

public class NPCInteraction : MonoBehaviour
{
    public float interactionDistance = 3f;
    public KeyCode interactKey = KeyCode.E;
    public GameObject interactionUI;

    private Transform player;

    void Start()
    {
        player = GameObject.FindWithTag("Player").transform;
        if (interactionUI != null)
            interactionUI.SetActive(false);
    }

    void Update()
    {
        float distance = Vector3.Distance(player.position, transform.position);

        if (distance <= interactionDistance)
        {
            if (interactionUI != null)
                interactionUI.SetActive(true);

            if (Input.GetKeyDown(interactKey))
            {
                Interact();
            }
        }
        else
        {
            if (interactionUI != null)
                interactionUI.SetActive(false);
        }
    }

    void Interact()
    {
        string[] lines = new string[]
        {
            "Hello there!",
            "This world has secrets hidden everywhere.",
            "Stay safe, traveler."
        };

        DialogueSystem.instance.StartDialogue(lines);
    }
}
```

  - `DialogueText` ‚Üí `DialogueSystem.dialogueText`
- Optional: Add an interaction UI (e.g., ‚ÄúPress E to talk‚Äù) and hook it to `NPCInteraction.interactionUI`
